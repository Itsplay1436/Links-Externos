PROJETO DE LINKS EXTERNOS

Este projeto oferece uma barra de navegação com links externos para sites de IAs (inteligências artificiais), além de uma breve explicação sobre suas funcionalidades e origens.

Além disso, são utilizados links externos, a tag <nav> para a barra de navegação e o uso das tags <details> e <summary> para a apresenteção dos alunos responsáveis pela criação dessa página.
